# Пакет клінічної валідації — КПТ-Клініка (VCT)

> Реалізує врядування зі [SPEC.md §8](SPEC.md). Мета: дати клініцисту-експерту змогу
> **перевірити й підписати** клінічний контент тренажера, перш ніж він використовується для
> навчання реальних терапевтів. Найбільший ризик навчального тренажера — **навчити неправильного**.

**Статус:** ⬜ очікує першої валідації · автор чернетки контенту — інженер (не клініцист).

## Як користуватися цим документом
Пройдіть розділи нижче. У колонці **Вердикт** позначте:

| Позначка | Значення |
|---|---|
| ✅ | Клінічно коректно, без змін |
| ⚠️ | Потребує зміни (впишіть, що саме, у «Коментар») |
| ❓ | Потребує обговорення / недостатньо даних |

Внизу кожного розділу — рядок підпису (ПІБ, спеціалізація, дата). Зміни з ⚠️ передаються
розробнику як задачі; після внесення — повторна перевірка пункту.

> ⚠️ **Джерела нижче наведені орієнтовно** (за міжнародними інструментами). Клініцист має
> підтвердити їх і, де можливо, послатися на **офіційно валідовані українські переклади** шкал.

## G.1-A. Шкали та пороги інтерпретації
Файли: [`app.js`](../app.js) (визначення `PACS`/`GAD7`/`PHQ9`), [`docs/ENGINE.md`](ENGINE.md).
Числа, які застосунок використовує для попереджень і висновків:

| Шкала | Діапазон | Поріг у застосунку | Підпис застосунку | Орієнтовне джерело | Вердикт | Коментар |
|---|---|---|---|---|---|---|
| **PACS** (крейвінг, алкоголь) | 0–30 | **≥ 15** → «ризик зриву» | «≥15 ризик зриву» | Flannery, Volpicelli, Pettinato, 1999 (Penn Alcohol Craving Scale) | | |
| **GAD-7** (тривога) | 0–21 | **≥ 10** → «помірна+» | «≥10 помірна+ тривога» | Spitzer et al., 2006 (пороги 5/10/15) | | |
| **PHQ-9** (депресія) | 0–27 | **≥ 10** → «помірна+» | «≥10 помірна+ депресія» | Kroenke, Spitzer, Williams, 2001 (пороги 5/10/15/20) | | |
| **PHQ-9 п.9** (думки про смерть/самоушкодження) | 0–3 | **будь-яка ненульова** → сигнал ризику + safety-логіка рушія | «⚠ виявлено суїцидальні думки» | стандартна практика скринінгу | | |

| Кейс | Подія | Коли | Очікувана правильна реакція стажера | Наслідок при пропуску | Вердикт |
|---|---|---|---|---|---|
| D2 «Подвійний тягар» | `safety_crisis` (ризик ≥2) | сесія **3** | активний скринінг суїцидального ризику; пряме чутливе запитання; безпекове планування | safety-override рушія → статус **crisis**, випадок закрито | |

## G.2. Калібрування симуляційного рушія (ENGINE_PARAMS)
Файл: [`src/engine/params.js`](../src/engine/params.js), формули — [`docs/ENGINE.md`](ENGINE.md).
Це **детерміновані коефіцієнти** (не ШІ). Зараз — **стартові дефолти**, потребують узгодження з експертами.

Ключові орієнтири, які треба підтвердити/відкалібрувати (а не кожне число):

| Підсистема | Поточний орієнтир | Що підтвердити | Вердикт |
|---|---|---|---|
| **Відмова від лікування** (§6.3) | добра сесія ~6–12% / погана ~25–37% відмови; головний важіль — **альянс**, не опір (контекст UA: пацієнт під тиском лишається й маніпулює) | Чи реалістична логіка «опір → маніпуляція, а не втеча»? Чи адекватні відсотки? | |
| **Зрив** (§6.2) | покращення готовності/тверезості реально знижує шанс зриву протягом курсу | Чи не домінує крейвінг абсолютно? Чи відчутний ефект терапії? | |
| **Safety-override** (§7) | пропущений ризик → `phq9 +3`, ескалація ризику, можлива криза; опрацьований → бонус альянсу +5, ризик −1 | Чи пропорційні «штраф» і «бонус»? | |
| **Критерії виписки** (§8) | готовність ≥81, PACS ≤10, PHQ-9 ≤10, ≥60 днів тверезості, 3 стабільні сесії поспіль | Чи це валідні критерії ремісії/готовності до завершення? | |
| **Подія життя / тригер** (§6.1) | базова ймовірність 0.35 за сесію, тяжкість 0.3–1.0 | Чи реалістична частота життєвих тригерів? | |

- [ ] За потреби — запропонувати конкретні значення коефіцієнтів (розробник внесе в `params.js`).
